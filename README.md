## 💡 Sobre o Projeto

Este projeto nasceu de uma ideia pessoal de criar uma interface cinematográfica com visual **cyberpunk/HUD** inspirada na franquia Velozes & Furiosos. O objetivo foi aprender programação front-end construindo algo visualmente imersivo e divertido.

A interface simula um painel de rastreamento futurista com 4 seções navegáveis, animações em CSS puro e lógica em JavaScript vanilla — sem frameworks, sem bibliotecas externas.

---

## 🎬 Funcionalidades

### 🏠 Página Início
- Carro favorito em destaque (**Nissan Skyline GT-R R34**)
- Animação de flutuação e rotação do carro
- Anéis orbitando ao redor com efeito neon
- Barra de especificações técnicas

### 🚙 Aba Carros
- Carrossel com **10 veículos icônicos** da franquia
- Navegação por setas e pontinhos
- Painel de dados técnicos de cada carro
- Animação de flutuação nas imagens

### 👥 Aba Personagens
- **8 personagens principais** com foto e perfil completo
- Efeito de scanner facial animado
- Tags de habilidades de cada personagem
- Carrossel com navegação

### 👁️ Olho de Deus
- Interface HUD inspirada no sistema do filme
- Globo central com anéis girando
- Painéis laterais com dados de missão
- Barras de sistema animadas (CPU, Sinal, Uplink)
- Terminal com logs em tempo real
- Relógio digital
- Barra de loading animada

### 🌐 Efeitos Globais
- Glow que segue o mouse no fundo
- Animação de entrada (fade) ao carregar a página
- Grade tecnológica no fundo
- Linha neon animada no cabeçalho
- Proteção das imagens contra clique direito e arrastar

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| **HTML5** | Estrutura e semântica |
| **CSS3** | Estilização, animações e layout |
| **JavaScript Vanilla** | Lógica, carrosséis e interatividade |
| **CSS Grid** | Layout principal |
| **CSS Custom Properties** | Paleta de cores centralizada |
| **Google Fonts** | Fontes Orbitron e Rajdhani |

> Nenhum framework ou biblioteca externa foi utilizado.

## 🚀 Como Rodar Localmente

1. Clone ou baixe o repositório
2. Abra a pasta no VS Code
3. Clique com botão direito no `index.html`
4. Selecione **"Open with Live Server"**

## 📚 O que Aprendi

Este foi meu primeiro projeto front-end completo. Durante o desenvolvimento aprendi:

- Estruturar páginas com **HTML semântico**
- Criar layouts com **CSS Grid**
- Usar **variáveis CSS** para paleta de cores
- Criar **animações** com `@keyframes`
- Manipular o **DOM** com JavaScript
- Trabalhar com **arrays e objetos** em JS
- Criar **carrosséis** sem bibliotecas
- Usar `setInterval` para animações em tempo real
- Organizar código em **arquivos separados**
